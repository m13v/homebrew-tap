# m13v/homebrew-tap

Homebrew tap for [m13v](https://github.com/m13v)'s tools.

## Usage

```sh
brew tap m13v/tap
brew install --cask claude-meter
```

Or in one line:

```sh
brew install --cask m13v/tap/claude-meter
```

## Casks

| Name | Description |
|------|-------------|
| [claude-meter](https://github.com/m13v/claude-meter) | Menu-bar app showing live Claude plan usage (5h + 7d limits) |
